OpenTHC API

A Proposal for Common API and Common Data Model

github.com/openthc/api

Data Models

The primary purpose of the OpenTHC API are these common data models, using the same words for the same attributes.

There are more than 100 software providers in the cannabis (and general horticulture) space, and just as many data-models and APIs. All for the same core data, with just minor differences. A common language makes interoperability easier for everyone.
